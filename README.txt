Welcome to Arondight's site. This direcotry is the elbuilds repository for Enterprise Linux. You can find some packages with new version here :)

Usage:

$ sudo rpm --import http://site.arondight.me/centos/pki/rpm-gpg/RPM-GPG-KEY-elbuilds
$ sudo yum install http://site.arondight.me/centos/7/x86_64/Packages/elbuilds-release/1.0-1/elbuilds-release-1.0-1.el7.elbuilds.noarch.rpm
$ sudo yum makecache
$ sudo yum update
